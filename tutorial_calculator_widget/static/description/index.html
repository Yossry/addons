This app is a tutorial in the form of code. The app will learn you how to create and use calculated fields in Odoo.

Learn how to add calculation functionalities to your fields

This app contains all the code you need to add the ability to do mathematical computations within fields.
All the code is documented inline with notes and extra information to understand how the calculation feature works.

Contains

  • New fields 'calculation' and 'calculation_int' on the 'product.template' model.
  • New fields 'calculation' and 'calculation_int' on the product and product variant form view.
  • Inline code explaining how the functionality works with links to the official code.
