This app is a tutorial in the form of code. The app will learn you how to create and use map views.

Learn how to create and use map views

This app contains all the code you need to add the Map view on (custom) models.
All the code is documented inline with notes and extra information to understand how the Map view works and what options you have.

Contains

  • New Map view on the quotations.
  • New field named "contact_address_complete" in the model "sale.order".
