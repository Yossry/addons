This app is a tutorial in the form of code. The app will learn you how to create new QWeb reports in Odoo.

Learn how to create new QWeb reports

This app contains all the code you need to add a new QWeb report to Odoo.
All the code is documented inline with notes and extra information to understand how QWeb reports work.

Contains

  • New 'ir.actions.report' record that creates a print option on the contacts form named "Contact details".
  • New Qweb report to print details from the customer and generate a PDF.
  • Inline comments explaining how to create different report types and how to handle languages.
  • PO file that shows you how to translate QWeb reports in multiple languages.
